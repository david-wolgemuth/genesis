# 2026-04-14 — Cycle 1: Scaffolding

## What I Did

Built the entire Genesis engine from scratch based on SPEC.md. This is the first PR — there was no existing codebase.

### Architecture

Followed the three-layer architecture faithfully:
- **world/** — Grid, terrain, star energy, diffusion, clocks
- **agents/** — Entity-component model, bonding engine, neighborhood interaction, tick loop
- **render/** — Snapshot events, isometric SVG renderer, dashboard HTML generator

### What Went Well

- The entity-component model is clean. Agents are just bags of components. A free alpha element and a bonded pair use the same struct — the difference is just whether the `bonds` set is empty.
- Conservation check works perfectly. The element census after every world tick caught zero violations across both test runs.
- The isometric renderer produces legible output — terrain depth is visible, activity hotspots glow, and the dashboard gives a quick overview of the run.
- Config loading via TOML is straightforward. The five elements, eight bond rules, and two catalysis rules all load correctly.

### What Was Hard

- Raw string literals with `#` characters in Rust. The `r#"..."#` syntax conflicts with HTML color codes like `#0a0a1a`. Had to use `r##"..."##` (double-hash delimiters) for strings containing color codes. Minor, but took a compile cycle to figure out.
- Getting the bonding probability right. Too high and everything bonds instantly in the first few ticks. Too low and nothing happens. Settled on 0.1 base probability with catalysis multiplier. This produces a nice gradual curve.
- Diffusion and bonding interact in a subtle way. Bonded agents don't diffuse (they're stuck in place), so bonding creates stationary clusters while free elements continue to drift. This is emergent and correct — but it means the spatial distribution changes over time in ways that affect future bonding rates.

### What I'd Do Differently

- The catalysis detection needs work. It requires all three agents (catalyst + both reactants) in the same cell, which is a rare three-body coincidence. Should probably expand to include catalysts in neighboring cells.
- Bond breaking never triggered — stability thresholds are too generous for the tidal pool's temperature range. Need to either tighten thresholds or add more temperature variation (vent proximity, day/night thermal cycling).
- The renderer shows final state only. Should capture intermediate snapshots to show the progression of bonding over time.

### Process Notes

- Wrote all code before first compile. This was risky but worked — only one compile error (the raw string issue).
- Ran with two different random seeds (42 and 123) to verify consistency. Both produced ~1,250 bonds and passed conservation. Good sign.
- The simulation runs fast — under 1 second for 5,000 ticks with 4,500 agents on an 80x40 grid. No performance concerns yet, but the grid is deliberately small for v1.
